Data Analysis Using Blueberry

By Patrick Zhang and Samuel Zhang

Introduction

  • Blueberry is a bio-sensing computer that is attached to an eyeglass, which collects biological signals to track physical and mental states.
  • We analyzed five key metrics from Blueberry: HbO (oxyhemoglobin), HbR (deoxyhemoglobin), flowActivity (a representation of changes in blood flow over a 10â€“20 second period), heartRate (beats per minute), and flowIntensity (a composite representation of blood flow changes and cardiovascular indicators such as heart rate).
  • The data came from a session where a person read The History of the Future: Oculus, Facebook, and the Revolution That Swept Virtual Reality by Blake J. Harris and took notes, with their notes, a CSV file, and a screen recording provided.
  • We used Google Colab, a software that allows us to run Python and work together, and a few libraries like pandas and Matplotlib to visualize the data.
